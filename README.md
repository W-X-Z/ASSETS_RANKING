# Assets Ranking

자산군별 연간/월간 수익률 비교 차트

## 기능

- 다양한 자산군(금, S&P500, 비트코인, 채권, KOSPI, Nasdaq, 부동산)의 수익률 비교
- YTD(Year to Date) 및 MTD(Month to Date) 기간 선택
- 자산군 필터링
- 인터랙티브 시각화
- 호버 효과로 상세 정보 확인

## 설치 및 실행
- bash

##의존성 설치
- npm install
##개발 서버 실행
- npm run dev
##API 서버 실행 (새 터미널에서)
- npm start
## 기술 스택
- D3.js: 데이터 시각화
- Express: API 서버
- Yahoo Finance API: 금융 데이터
- Parcel: 번들러
